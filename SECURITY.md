# Security Policy

## Supported Versions

`toLocal` is pre-release. Security fixes currently target the latest branch.

## Reporting a Vulnerability

Please do not open a public issue for a suspected vulnerability. Use GitHub's
private vulnerability reporting for this repository once enabled.

Include:

- A description of the issue and its impact.
- Reproduction steps or a minimal fixture.
- The affected extension version and Chrome version.
- Any suggested mitigation.

Do not include sensitive page content, credentials, or private URLs.
